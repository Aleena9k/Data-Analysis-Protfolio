# UK Retailer - Ecommerce Dataset Analysis
This project analyzes an E-Commerce dataset containing transactions from a UK-based online retailer between 2010 and 2011. The dataset includes all the invoices generated during this period for both wholesale and retail customers, primarily selling giftware and homeware products.
